## Weekly Checklist (45 min total)

### MONDAY — GSC + Social Check (5 min)
- [ ] Open Google Search Console

- [ ] Note any new terms you don't explicitly target

- [ ] Run Answer Socrates on 3 keywords (5 min)
  - Example: "golf betting," "Nassau," "Skins"
- [ ] Harvest Google autocomplete (5 min)
  - Type seed keyword, add space + letter A-Z
- [ ] Extract PAA questions for 2 keywords (3 min)
  - Use SEO Minion or manual Google search
- [ ] Review notes, flag 3+ similar questions → add to pipeline

### AD-HOC (25 min as needed)
- Whenever you find 3+ variants of same question
- Assess: Does Stick have content? (Yes = internal link; No = new task)
- Assign content task if gap identified
- Update pipeline status

## Questions to Ask When You Find a Keyword

1. **Is this question appearing multiple places?** (AC, PAA, Reddit, GSC)
   - YES → Create content
   - NO → Maybe internal link

2. **Does Stick have content answering this?**
   - YES → Add internal link, maybe expand FAQ
   - NO → Assign new content task

3. **When will people search for this?** (Seasonal?)
   - March-April → Masters window
   - June → Majors window
   - September → Fall tournaments
   - → Publish 2-3 weeks before

4. **Who is searching?** (New player? Advanced? Competitor scout?)
   - New player → Blog post, beginner-friendly
   - Advanced → Guide expansion, settlement math
   - Competitor scout → Doesn't matter, target all

5. **What's the intent?** (Learn? Compare? Troubleshoot?)
   - Learn → Blog post, educational
   - Compare → Comparison content (Sixes vs Split Sixes)
   - Troubleshoot → FAQ, settlement guides
