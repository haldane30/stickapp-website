## Decision Framework: When to Act on Data

| Signal | Action |
|--------|--------|
| **Page ranking 5-20 with high impressions** | Expand content, add FAQ, refresh title. This is striking distance — a small push could reach page 1. |
| **Page ranking 1-5** | Protect it. Add internal links from new content. Don't make major changes that could disrupt ranking. |
| **Page indexed but 0 impressions after 4 weeks** | Check: is the title targeting the right keyword? Is there a cannibalization issue? Does the page need more internal links? |
| **New keyword appearing in "emerging" list** | Check if we have content for it. If not, add to Pipeline as identified. If so, optimize the existing page. |
| **Competitor publishes new content for our keyword** | Don't panic. Check if our content is still better. If not, expand. If yes, monitor for 2-4 weeks. |
| **CTR below 2% for a page with impressions** | Title or meta description isn't compelling. Rewrite to be more specific and include a hook. |
| **High clicks but high bounce rate** | Content isn't matching search intent. The page may need restructuring or the H1 may be misleading. |
