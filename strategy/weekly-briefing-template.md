# Weekly Intelligence Briefing — Stick Golf

> **Purpose:** A fresh briefing generated at the start of each session (or weekly). This is not a static document — Claude writes it fresh each time based on current research, pipeline status, and external signals.
>
> **How to trigger:** Ask Claude: "Run the weekly briefing" or "What should we focus on this week?"
>
> **What Claude does when this runs:**

---

## Briefing Checklist (Claude follows these steps)

### 1. Pipeline Check
- Read `strategy/content/content-pipeline.md`
- What's the next piece to write? Is it researched and outlined?
- Are there 3-4 pieces in "Researched + Outlined" status? If not, that's job #1.

### 2. Calendar Check
- Read `strategy/content/content-calendar.md`
- Is any content due this week or next?
- Is there a golf moment within 3 weeks that we need to publish for? (Must publish 2+ weeks before moment for indexing.)

### 3. Scoreboard Check
- Read `strategy/seo/seo-scoreboard.md`
- Has Justin provided new Search Console data?
- Any striking-distance keywords to act on?
- Any emerging keywords that suggest new content opportunities?

### 4. Trending Golf Search Scan
Using web search, check:
- Search Google for "golf betting" and note People Also Ask questions — any new ones we're not covering?
- Search for "best golf games" / "fun golf games with friends" — who ranks and is their content good?
- Search for Stick's primary keywords (nassau golf, wolf golf game, golf skins game) — has our ranking changed? Has a competitor published anything new?
- Check if any major golf news is trending (tournament week, rule change, viral golf moment) that creates a content opportunity.

### 5. Competitor Pulse (2-minute check)
- Quick look at beezergolf.com — any new blog posts?
- Quick look at skinsapp.com — any site changes?
- Are any new apps showing up in "golf betting app" searches?

### 6. "What Justin Should Know"
Surface insights from the research above:
- Any new content opportunities discovered?
- Any threats or shifts in the landscape?
- Any product insights (features golfers are asking about that Stick should consider)?
- Any time-sensitive opportunities to pounce on?

### 7. Recommended Focus
Based on all of the above, recommend:
- What to work on THIS session (1-2 items)
- What to prepare for NEXT session
- Any urgent items (calendar deadlines, competitive threats)

---

## Output Format

Write the briefing as a conversational summary to Justin — not a checklist. Lead with the most important/urgent thing. Be direct. If nothing is urgent, say so and recommend the highest-impact work from the pipeline.

Example tone:
> "Nothing urgent this week — no calendar deadlines until Masters prep in late March. The pipeline has 'Golf Betting for Beginners' as next up and it's fully outlined. I did a quick search scan and noticed [insight]. My recommendation: let's draft the beginners piece today and knock out the Wolf 3-player expansion as a quick win alongside it."

---

## After the Briefing

If the briefing reveals new insights:
- Add new content opportunities to the Pipeline (with source and rationale)
- Update the Scoreboard if new ranking data was found
- Log any "What Justin Should Know" items in CLAUDE.md under a dedicated section
